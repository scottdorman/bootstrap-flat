Bootstrap Flat

A simple flat theme for Bootstrap 3.

Download latest View on GitHub

Heads up! Downloads are pulled directly from the latest committed code on GitHub, and as a result our docs may at times be out of sync.

Created from a need.

While Bootstrap 3 does away with the glossy and gradient characteristics of earlier versions, making it mostly a flat design, it doesn't completely embrace the "modern" design elements (inspired by Microsoft's Modern Design Language of Windows Phone and Windows 8) and still includes soft, rounded edges on many of the UI elements.

Although there are already several other "flat" or "modern" skins for Bootstrap, they aren't built for Bootstrap 3. They are also completely custom Bootstrap implementations and in most cases include their own, modified, version of bootstrap.css making it difficult to upgrade to newer versions of Bootstrap.

Bootstrap-flat aims to change that. It's built on top of Bootstrap 3 and uses Bootstrap's recommended method of providing "light" customizations. In other words, these customizations are pure CSS.

Highly encouraged for sharing.

Free to use for any projects you have in mind, although it would be nice if you gave credit.
